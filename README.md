# Σύστημα Διαχείρισης Χρηστών (CRUD)

Single Page Application (SPA) για τη διαχείριση χρηστών, υλοποιημένη με  HTML, CSS και JavaScript.

## Χαρακτηριστικά
* **CRUD Λειτουργίες**: Δημιουργία, Ανάγνωση, Ενημέρωση, Διαγραφή χρηστών.
* **Live Validation**: Εμφανίζει μηνύματα σφάλματος ή επιτυχίας καθώς πληκτρολογείτε.
* **Αυτόματη Μορφοποίηση**: Τα ονόματα μετατρέπονται αυτόματα σε Κεφαλαία Ελληνικά χωρίς τόνους.
* **Μοναδικό Email**: Το σύστημα δεν επιτρέπει διπλότυπα emails.
* **Ταξινόμηση**: Κλικ στους τίτλους του πίνακα (Όνομα, Email, Ημερομηνία) για αύξουσα/φθίνουσα ταξινόμηση.
* **Responsive Design**: Προσαρμόζεται σε οθόνες κινητών.
